## Project Overview

This project is created as part of the Digital Literacy course. The main aim of this project is to understand how to use digital tools in a proper and safe way. It also helps in learning how to create a professional online presence and communicate effectively in the digital world.

In this project, I have completed different tasks related to digital awareness, online platforms, communication, and cyber safety.

## Tasks Included

### Task 1 – Digital Literacy Infographic
In this task, I created an infographic to explain the concept of digital literacy, safe internet practices, and useful tools for students.

### Task 2 – Digital Portfolio
In this task, I created and updated my profiles on platforms like GitHub and LinkedIn to build my professional online presence.

### Task 3 – Platforms Exploration
In this task, I explored coding platforms and created a Google Form quiz related to digital literacy.

### Task 4 – Email Etiquette
In this task, I wrote professional emails and created a checklist for proper social media usage.

### Task 5 – Cybercrime Awareness
In this task, I studied cybercrime and created a case study along with prevention tips to stay safe online.

## Conclusion

This project helped me understand the importance of digital literacy and how to use digital platforms in a responsible and effective way.
